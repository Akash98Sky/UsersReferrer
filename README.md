# UsersReferrer

### 1. Setup Environment
Create a .env file with following content
```.env
PORT=
MONGODB_URI=
LOG_LEVEL=debug
```

### 2. Install all npm packages
```bash
$ npm install
```

### 3. Run the application
```bash
$ npm start
```
